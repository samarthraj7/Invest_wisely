# Sample decks

Drop a real pitch deck here to validate the pipeline, e.g.:

```
sample_decks/acme_seed_deck.pdf
sample_decks/acme_seed_deck.pptx
```

Then run from `backend/`:

```bash
python analyze_deck.py ../sample_decks/acme_seed_deck.pdf
```

> A placeholder deck is generated automatically the first time you run the CLI
> with `--demo`, so you can exercise the full flow before adding your own file.
